LITA Forum - November 2017

Launching your own Linked Data Application

Using the open-source library linked-data BIBCAT platform, this introductory program provides participants with the knowledge to start planning and building their own lightweight RDF-based applications with vocabularies like BIBFRAME and Schema.org.

BIBCAT, an open-source middleware for constructing RDF applications, is currently used in a number of projects including a consortium linked-data MARC-to-BIBFRAME publishing effort, a regional DP.LA hub for Colorado and Wyoming, and as the backbone for a new Academic Concierge mobile app for higher education students at midsize public university.
